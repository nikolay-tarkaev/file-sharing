1. Создать базу данных, или использовать ранее созданную;
2. В файлах sql в первой строке указазать имя базы данных, сохранить изменения;
3. Импортировать файлы sql;
4. Открыть файл application/bootstrap.php;
5. В строке 15, заполнить данные базы данных (пример в комментарии);
6. Папка web должна быть корневой директорией веб сервера;
7. Версия PHP 5.5.0 или выше.

Пример: upload.tarkaev.pp.ua
