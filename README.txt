2. Открыть файл script.sql, и в первой строке указазать имя базы данных, сохранить изменения;
3. Импортировать файл script.sql;
4. Открыть файл application/bootstrap.php;
5. В строке 15, заполнить данные базы данных (пример в комментарии);
6. Папка web должна быть корневой директорией веб сервера.
